# devenvtools

Small shell utilities for local development environment setup.

## Scripts

- `chatgpt-theme.sh` controls ChatGPT for macOS appearance behavior.
- `setup-macos-ls-style-modular.zsh` installs a modular zsh `ls`/`eza` style setup. Despite the historical filename, the script now supports macOS and Linux.
- `setup-zsh-proxy-functions-modular.zsh` installs modular `proxy_on`, `proxy_off`, and `proxy_status` helpers for zsh.

## Usage

Run each script with `--help` to see its options:

```sh
./chatgpt-theme.sh --help
./setup-macos-ls-style-modular.zsh --help
./setup-zsh-proxy-functions-modular.zsh --help
```
